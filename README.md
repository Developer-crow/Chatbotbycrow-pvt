# Chatbotbycrow-pvt

crow_10} chatbot used the open ai api to create a chat bot.

## installation

get your api key from openai and paste the key into the ```api/.emv``` files

used the pack manager [npm](http://nodejs.org/) to install the required package.

```bash
cd api
npm i
```

## Usage

```bash
cd api
node .
```
